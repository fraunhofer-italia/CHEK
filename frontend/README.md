## Project Setup

```sh
npm install
```

### Local: Compile and Hot-Reload for Development

```sh
npm run dev
```
### Expose to network: Compile and Hot-Reload for Development

```sh
npm run host
```

### Compile and Minify for Production

```sh
npm run build
```
